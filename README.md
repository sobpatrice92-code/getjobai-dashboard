# 🚀 GetJobAI Dashboard

Dashboard professionnel pour la recherche d'emploi automatisée avec IA.

## ✨ Features

- 🎨 **UI UX PRO MAX** - Design system professionnel
- 📊 **175 offres** - Multi-sources (LinkedIn, Job Bank, Talent.com, Charity Village)
- 🤖 **6 Agents IA** - Automatisation complète
- 📤 **Candidatures auto** - Smart Apply intégré
- 💌 **Humanizer** - Lettres indétectables par IA

## 🚀 Déploiement

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://getjobai.streamlit.app)

## 🛠️ Tech Stack

- **Frontend:** Streamlit + UI UX PRO MAX
- **Backend:** Python 3.11+
- **Database:** Supabase
- **AI:** OpenAI GPT-4o

## 📦 Installation Locale

```bash
git clone https://github.com/sobpatrice92-code/getjobai-dashboard
cd getjobai-dashboard
pip install -r requirements.txt
streamlit run streamlit_app.py
```
